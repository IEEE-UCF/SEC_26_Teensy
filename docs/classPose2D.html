Detailed Description

Class for handling 2D objects.

This class can represent position, acceleration, or velocity. Users must track the type of value stored. For example, fixTheta() should be used on a position vector but not on a velocity vector. Many functions return Pose2D, allowing method chaining.

Aldem Pido, Rafeed Khan
