Detailed Description

Manages multiple VL53L0X Time-of-Flight distance sensors using an I2C multiplexer. This class handles the initialization, update, and data retrieval for an array of VL53L0X sensors, each connected to a different channel of an I2C multiplexer.

bool TOFHandler::Begin ( )

Initializes all configured VL53L0X sensors. Iterates through each sensor, selects its I2C multiplexer channel, sets the I2C bus (Wire1), and initializes the sensor.
