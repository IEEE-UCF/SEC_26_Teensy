class  RGBHandler
 Manages RGB LED strip effects across multiple sections. This class handles initialization, updates, and control of WS2812-based LED strips. It allows dividing the strip into logical sections, each capable of displaying independent effects like solid color, pulsing, or streaking. It also supports global brightness control. More...
