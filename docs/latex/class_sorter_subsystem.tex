\doxysubsection{Detailed Description}
Manages an object sorting mechanism using TOF, Hall sensors, a servo, and a motor. This subsystem integrates multiple sensors and actuators to detect objects, identify their properties (e.\+g., magnetic), and sort them using a servo-\/driven mechanism and a transfer motor.
