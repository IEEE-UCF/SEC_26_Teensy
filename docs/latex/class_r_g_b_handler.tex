void RGBHandler\+::\+Update (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Main update loop for all LED effects. This function should be called repeatedly (e.\+g., in the Arduino {\ttfamily loop()} function). It iterates through all defined sections. If a section has an active effect (PULSE or STREAK) and enough time has passed since its last update (determined by {\ttfamily effect\+Speed}), it calls the respective update function ({\ttfamily update\+Pulse} or {\ttfamily update\+Streak}). If any section was updated, {\ttfamily leds.\+show()} is called to refresh the LED strip, but not more often than approximately 60 times per second (every 16ms) to save processing time.
