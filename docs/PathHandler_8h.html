Classes

class  PathHandler
 Manages a sequence of waypoints for robot navigation. This class allows for adding waypoints, executing a path defined by these waypoints, handling timeouts, and checking if waypoints have been reached. It interacts with a VectorRobotDrivePID object to control the robot's movement and to get its current pose. More...
